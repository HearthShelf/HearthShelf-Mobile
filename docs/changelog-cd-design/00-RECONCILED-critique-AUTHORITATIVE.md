## RECONCILED CANONICAL DECISIONS (authoritative for the implementer)

- `changelog_items` (id, `changelog_id` FK, section, text, sort_order). Drop the denormalized `tags TEXT` column.

**Canonical sections (exactly 6):** `feature | fix | change | docs | breaking | other`. No `added`/`removed`. `removed`-verb commits → `change`. Display order everywhere: **breaking, feature, fix, change, docs, other**. Labels: BREAKING / ADDED / FIXED / CHANGED / DOCS / NOTE.

{ "product":"HearthShelf-Mobile", "version":"0.1.0",

No `changelog` string field. No `addon_name`. No back-compat alias.
