Why this menu

Floating, but not mysterious.

The dock reuses the app’s existing floating-glass language and gives it a foldable-specific placement preference. It never permanently narrows a pane, and drag has visible button alternatives.

Library rule

Connected first. Focus on demand.

Selecting a cover updates the right leaf without losing grid position. “Open full book page” promotes that selection to the full two-leaf detail route when the listener wants depth.

Player rule

The app still feels like itself.

Queue count, title/author header, whole-book progress, cover, chapter scrubber, five-part transport, and customizable actions remain in their current order. Only the former sheets move beside it.
