Screen 05 · Stats & Clubs

The in-house teardown

The consultant found and fixed a genuine bug (heatmap month labels computed but never drawn) and added pull-to-refresh, a goal ring, and club unread badges. All correct. But "tappable heatmap days" at ~12px is a new accessibility problem, and a data visualization with no non-visual path fails a11y outright.

Keep the fixed labels and the visual. Make the whole column (a week) the tap target with a tooltip that reveals the specific day, so a 12px cell never has to be hit precisely. Add a "View as list" toggle that exposes the same data as an accessible table for screen readers and motor-impaired users.
